# Demo Video Recording Checklist

Use this checklist to ensure your demo video is production-ready.

## Pre-Recording

- [ ] Production URL tested and working
- [ ] Test receipt image ready (clear, readable)
- [ ] Screen recording software installed and tested
- [ ] Browser cleaned (incognito mode, no extensions)
- [ ] Audio levels tested (if narrating)
- [ ] Recording area set (browser window)
- [ ] Quiet environment secured

## Recording Setup

- [ ] Browser window sized appropriately
- [ ] Zoom level set to 100%
- [ ] Page loaded: `/test`
- [ ] Upload zone visible
- [ ] No personal information visible
- [ ] Recording started

## During Recording

- [ ] Opening (5s) - Page shown, upload zone visible
- [ ] Upload (3s) - Receipt uploaded successfully
- [ ] Processing (5s) - Status shows, spinner visible
- [ ] Results (12s) - Data displayed, scrolled through
- [ ] Closing (5s) - CTA/logo shown
- [ ] Total duration: ~30 seconds
- [ ] No errors or glitches captured

## Post-Recording

- [ ] Video reviewed for quality
- [ ] Beginning/end trimmed
- [ ] Text overlays added (if needed)
- [ ] Audio synced (if separate recording)
- [ ] Exported in high quality (1080p+)
- [ ] File size reasonable (<50MB)

## Upload & Distribution

- [ ] Video uploaded to platform (Loom/YouTube/Vimeo)
- [ ] Shareable link obtained
- [ ] Link tested (opens correctly)
- [ ] Email template prepared
- [ ] 3 prospects identified
- [ ] Ready to send tomorrow

## Final Verification

- [ ] Video shows value clearly
- [ ] No technical issues visible
- [ ] Professional presentation
- [ ] Clear call-to-action
- [ ] Ready for prospects
