## Detailed Action Steps

### Step 1: Setup (Before Recording)

1. Open browser in incognito/private mode (clean state)
2. Navigate to: `https://your-dashboard.vercel.app/test`
3. Wait for page to fully load
4. Resize browser window to show full page
5. Hide browser bookmarks bar (optional, cleaner look)
6. Close unnecessary tabs

### Step 2: Recording Setup

1. Start screen recording
2. Set recording area to browser window
3. Optional: Add webcam overlay (if personal touch desired)
4. Test audio levels (if narrating)

### Step 3: Recording

1. **Countdown**: "3, 2, 1..."
2. **Start recording**
3. Follow script above
4. **End recording** after closing

### Step 4: Post-Production (Optional)

1. Trim beginning/end (remove countdown, dead time)
2. Add text overlay:
   - "DocFlow - Automated Receipt Processing"
   - "Try it: your-dashboard.vercel.app/test"
3. Add background music (optional, keep it subtle)
4. Add fade transitions
5. Export in high quality (1080p minimum)

## Troubleshooting

### Recording Issues

**Issue**: Screen recording laggy
- **Solution**: Close other applications, use hardware acceleration

**Issue**: Audio out of sync
- **Solution**: Record audio separately, sync in post-production

**Issue**: Browser crashes during recording
- **Solution**: Use lighter browser (Chrome), close extensions

### Demo Issues

**Issue**: Upload fails during recording
- **Solution**: Pre-test, have backup receipt ready

**Issue**: Processing takes too long
- **Solution**: Use smaller image, or speed up in post-production

**Issue**: Results don't display correctly
- **Solution**: Test beforehand, use known-good receipt
